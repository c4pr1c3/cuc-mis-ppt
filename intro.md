## 移动互联网全生命周期视角

| 阶段             | 攻防知识点和技能                   |
| :-               | :-                                 |
| 加入无线网络之前 | **嗅探** 、⼊侵（⽹络）、钓鱼、DoS |
| 加入无线网络之后 | **嗅探** 、⼊侵（终端）、MITM      |
| 使用蜂窝数据网络 | **嗅探** 、MITM                    |
| 使用移动应用     | **嗅探** 、篡改、⼊侵（终端&云端） |
